# Nurtura

Trusted pregnancy guidance app with Flutter frontend and Node.js API.

## Features

- Ask AI chat with multiple threads and RAG knowledge retrieval
- Pregnancy week guidance, diet, emergency info, and appointments
- Local PostgreSQL backend with embedded database support

## Run locally

```powershell
# API
cd api
npm install
npm start

# Flutter app
cd ..
flutter pub get
flutter run -d chrome
```
